# BGM 搜索关键词参考

> 在剪映中搜索 BGM 时，按情绪类型使用以下关键词

## 按情绪分类

| 情绪 | 中文关键词 | 英文关键词 |
|------|-----------|-----------|
| 悬疑/震惊 | 悬疑 开场 氛围 | mystery ambient intro |
| 宏大/史诗 | 非洲 大自然 史诗 / 史诗 纪录片 | epic documentary nature |
| 紧张/对峙 | tension subtle / 紧张 暗流 | tension underscore |
| 温暖/日常 | acoustic guitar warm / 非洲 原声 温暖 | warm acoustic african |
| 荒诞/幽默 | sneaky playful / 轻松 诙谐 | quirky playful |
| 行动/军事 | military patrol tension / 军事 巡逻 | action patrol tension |
| 安静/沉思 | piano ambient ending / 黄昏 安静 | ambient piano reflective |
| 悲壮/沉重 | 悲壮 纪录片 / solemn documentary | solemn emotional |

## 全片统一 BGM 推荐

如果不想逐场景配乐，找一首覆盖全片的 BGM：
- **非洲/热带题材**：`非洲 纪录片 配乐` / `african documentary soundtrack`
- **灾难/事故题材**：`灾难 纪录片 配乐` / `disaster documentary music`
- **军事/谍战题材**：`谍战 悬疑 配乐` / `spy thriller underscore`
- **历史/人物题材**：`历史 人物 纪录片` / `historical biography soundtrack`

## 使用原则

- BGM 音量压低到配音的 **20-30%**
- 不要在配音说话时用强节奏的 BGM
- 场景转折处可以用 BGM 变化辅助叙事
