# 第9步：提取台词

从分镜脚本中逐段提取纯台词，格式化后输出为剪映可直接导入的文稿文件。

**输入**：每个分镜的「台词」文本

**处理规则**：
- 逐分镜提取台词内容
- 去除★金句标注等非台词标记
- **格式化规范**：
  - 一句一换行（换行=分句信号）
  - 无标点符号（句号、逗号、冒号、引号等全部移除，用换行代替）
  - 保留书名号《》、引号「」等特殊标识（如船名「加利福尼亚人」）
- 保持分镜顺序输出

**输出**：保存到 `<projects_dir>/[项目名]/文案/字幕文稿.txt`

**示例**：
```
1912年4月14日深夜
泰坦尼克号最后的冰山警告淹没在一堆付费电报里
首席电报员杰克菲利普斯已连续工作了十几个小时
...
```
