# 第7步-生成配音 评审检查清单

## 默认级别：🔴 L3（深度验证）

- [ ] L1 全部检查项（文件存在、数量正确、时长>0、格式正确）
- [ ] L2 检查：波形特征、文件大小与时长成比例
- [ ] ffprobe 全量提取（codec/sample_rate/channels/bit_rate/duration）
- [ ] 参数一致性：采样率/声道数/比特率统一
- [ ] 特殊处理验证：★/☆金句标记已正确删除（★及其后的文本未被读出）
- [ ] 标点处理正确（省略号、引号等）
- [ ] 逐段对比台词与原始脚本，确认无遗漏/多余
- [ ] 语调检查：抽查情绪高潮段落，语调匹配台词内容（避免读稿感）
- [ ] 生僻字检查：多音字、专业名词、人名地名发音正确
- [ ] 语速检查：抽查部分段落计时，参考 3-5 字/秒
- [ ] 参数边界：采样率 16000-44100Hz，比特率 64-320kbps，各段参数统一
- [ ] 输出路径正确（素材/配音/scene0x_描述.mp3）
