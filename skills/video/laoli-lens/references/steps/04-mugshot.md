# 第4步：生成角色定妆照

**调用**：`laoli-imagine` skill

- **输入**：`核心角色设定`中的「定妆照提示词」
- **输出**：竖屏9:16图片，保存到 `素材/定妆/ref_角色ID.jpg`
- **数量**：每个核心角色1张
- **生成顺序**：按`核心角色设定`的角色顺序逐个生成

> **不需要定妆照的角色**：群众/配角（无角色设定）

**批量模式说明**：多个角色时 agent 可能会走 `laoli-imagine` 的 batch 模式。请将 `batch.json` 生成到 `素材/定妆/` 目录下（而非项目根目录），使得 `image`、`ref` 等路径自然以 `素材/定妆/` 为基准解析。
