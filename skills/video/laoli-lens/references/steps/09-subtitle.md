# 第9步：提取台词

从分镜脚本中逐段提取纯台词，格式化后输出为剪映可直接导入的文稿文件。

**输入**：每个分镜的「台词」文本

**处理规则**：
- 逐分镜提取台词内容
- 去除 ★ 符号本身，保留其后的金句文本
- **格式化规范**：
  - 一句一换行（换行=分句信号）
  - 无标点符号（句号、逗号、冒号、引号等全部移除，用换行代替）
  - 保留书名号《》、引号「」等特殊标识（如船名「加利福尼亚人」）
- 保持分镜顺序输出

**输出**：保存到 `<projects_dir>/[项目名]/文案/字幕文稿.md`

**示例**：
```
1912年4月14日深夜
泰坦尼克号最后的冰山警告淹没在一堆付费电报里
首席电报员杰克菲利普斯已连续工作了十几个小时
...
```

## ✅ 本步完成条件

字幕文稿保存完成后，必须通过 Task tool 调用 `laoli-lens-reviewer` 进行评审（默认 L2：逐段对比准确性）。仅当评审返回 **✅ 通过** 后，方可告知 Owner 本步完成、进入下一步。**未完成评审 = 本步未完成。**
