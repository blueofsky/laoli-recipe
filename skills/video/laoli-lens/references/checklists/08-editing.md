# 第8步-视频剪辑 评审检查清单

## 默认级别：🔴 L3（深度验证）

- [ ] L2 全部检查项
- [ ] ffprobe 全量提取时长
- [ ] 裁剪完整性：裁剪后画面内容完整，关键视觉元素无偏移缺失
- [ ] 音视频时长误差 < 阈值
- [ ] 变速系数 < 3.0（setpts）
- [ ] 裁剪后分辨率记录正确
- [ ] 无水印帧抽查
- [ ] 音效文件完整（每个分镜对应一条）
- [ ] 音效变速后时长 ≈ 配音时长
- [ ] atempo 在 0.5-2.0 范围内，超出已链式处理
- [ ] atempo 值为 setpts 的倒数（互为倒数校验）
- [ ] 输出报告（视频剪辑.md）格式正确，数据完整
- [ ] 原始文件未被修改（素材/视频/原始/ 中的文件未动）
