# 第5步-生成分镜图 评审检查清单

## 默认级别：🟡 L2（标准评审）

- [ ] L1 全部检查项（文件存在、数量正确、大小合理）
- [ ] 完整性校验：文件正常生成（非0KB）
- [ ] 硬伤检测：角色面部扭曲、五官错位、多手指等明显AI缺陷
- [ ] 锚点一致性：角色与第2步视觉锚点一致（发色、脸型、配饰）
- [ ] 风格固化：风格后缀生效，画面质感符合方案视觉风格
- [ ] 历史/文化逻辑：道具/背景/服饰符合时代设定
- [ ] 角色区分度：多角色时面孔有足够区分度，避免克隆脸
- [ ] 光影情绪匹配：画面亮度和色调符合情绪氛围
- [ ] 参考图还原度：角色五官、神韵与定妆照参考图一致
- [ ] 叙事张力：画面准确传达了该分镜台词的核心情绪
- [ ] 构图与美感：无穿模或不自然的构图截断
- [ ] 画面中无多余文字/水印/Logo（扫描边缘和角落）
- [ ] 参考图权重适中：构图未被参考图锁死，核心元素符合 Prompt 描述
