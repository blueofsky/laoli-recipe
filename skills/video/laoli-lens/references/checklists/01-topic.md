# 第1步-选题大纲 评审检查清单

## 默认级别：🟢 L1（快速通过）

- [ ] 产出文件存在且路径正确（创作底本.md）
- [ ] 文件格式正确
- [ ] 必填字段完整：方案、亚型、选题标题、核心人物、核心冲突、3秒钩子、叙事引擎、节奏段
- [ ] 文件头部格式正确：`> 方案：<文件名>` + `> 亚型：<亚型名>`
- [ ] 亚型值有效：取自方案配置「亚型选择」表中的四种亚型之一
- [ ] 产出格式校验：对照产出格式逐字段校验
- [ ] 方案铁律合规：对照 profiles 风格铁律逐条校验
- [ ] 步骤边界校验：节奏段规划属于第1步职责；分镜细节属后续步骤，不做阻塞检查
- [ ] 必须有"抉择时刻"，不是事件流水账
- [ ] 开场钩子必须可视觉化（具体画面，非抽象概念）
- [ ] 情绪曲线有高点有低谷
- [ ] 每个节奏段以方案定义的段落标题命名
