# 第6步-生成视频 评审检查清单

## 默认级别：🟡 L2（标准评审）

- [ ] L1 全部检查项（文件存在、数量正确、时长>0、格式正确）
- [ ] 目视抽查关键帧：内容与分镜/Prompt 匹配
- [ ] 运动幅度合理，主体轮廓稳定
- [ ] 无画面闪烁或跳帧
- [ ] 运动描述匹配预期（不是纯静止画面）
- [ ] 分辨率符合预期
- [ ] 循环感检测：对比首尾帧，确认有实际运动变化（非循环/静止）
- [ ] 主体稳定性：主体轮廓全程保持一致，无变形/消失/重现
- [ ] 运动自然度：人物关节运动协调，表情变化自然，无机械感
- [ ] 输出路径正确（素材/视频/原始/）
