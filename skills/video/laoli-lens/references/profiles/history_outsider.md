# 方案配置 - 历史局外人

## 方案设定
*本方案专为"历史的局外人"短视频系列设计，聚焦宏大历史叙事中被忽略、被裹挟的关键小人物，通过他们的瞬间抉择，折射时代巨变与人性的复杂光谱。*

- **核心调性**：深沉、悲悯、充满命运感。拒绝脸谱化的英雄或恶人叙事，追求在历史褶皱中发现人性的颤抖与微光。
- **价值主张**：历史由胜利者书写，但由无数"局外人"的代价构成。我们关注代价，理解复杂，从而更透彻地看清权力与人性的游戏规则。
- **受众画像**：20-40岁，对历史、权谋、人性故事有浓厚兴趣，不满足于史实复述，渴望获得情感共鸣与认知颠覆的泛知识群体。
- **一句话定位**：给普通人的权力游戏说明书。

## 视觉风格设定
*本系列追求电影级的历史厚重感与戏剧张力，视觉服务于叙事，营造沉浸与压迫并存的观看体验。*

- **核心视觉标签**：
  - **艺术风格**：Cinematic, realistic oil painting style, heavy historical texture
  - **画质与光影**：8K resolution, ultra-detailed, Rembrandt lighting with strong chiaroscuro (伦勃朗式光影）但主角面部须有独立主光（key light）补亮，确保表情可辨， cinematic film grain
  - **色彩**：整体偏向中低饱和度，以棕、黑、金等厚重色调为基底，但在关键叙事元素（主角面部、历史小物件）上保留局部色彩锚点，避免全片暗沉。
  - **时代适应性**：视觉风格应随题材时代背景做合理调整。古代/中世纪题材以油画质感、伦勃朗光影为主；近现代题材（工业革命后）可适度转向摄影级纪实风，保留低饱和度+强光影对比的核心原则，但画面质感更贴近当时的摄影/影像记录风格。禁止使用明亮的数码感色彩。
- **视觉锚点与构图**：
  1. **主角特写**：必须强化面部特写，尤其是眼部，以传达恐惧、犹豫、决绝或空洞。面部是情绪的战场。
  2. **环境对比**：利用光影强烈区分"权力中心"（光亮处）与"局外人徘徊之地"（阴影处）。
  3. **关键符号**：每期视频须有一个核心的、可操作的"历史小物件"特写，作为命运的物理象征。古代题材示例：一道密旨、一把钥匙、一杯毒酒、一块玉玦；近现代题材示例：一枚密封圈、一张电报稿、一份签字的发射批准书、一张股市行情纸。核心原则：这个物件必须承载"命运的转折点"——它在主角手中或眼前被做出、被使用、被忽视的瞬间，就是历史分叉的时刻。
  4. **构图**：多使用框架式构图、不对称构图，营造压抑或动荡感。定妆照采用半身像、深色纯背景，突出人物状态。
- **动态运镜**：
  - 以缓慢的推、拉、摇为主，模仿"凝视"与"审视"，营造沉浸感和时间流逝感。
  - 关键动作（古代示例：开门、举剑、落笔；近现代示例：按下按钮、撕毁文件、签署批准书）可辅以细微的镜头震动或速度变化，强化冲击力。
  - 禁止无意义的快节奏闪烁剪辑。

## 叙事设定

### 叙事人设
- 叙述者是一名冷静、抽离但心怀悲悯的"历史观察者"。语调沉稳，不煽情，但字句充满力量。
- 与观众的关系是"分享洞察的老友"，而非"灌输知识的老师"。

### 亚型选择
*"历史的局外人"统一遵循三条底层规则：①宏大历史现场 ②微小具体的行为 ③极具讽刺性的个体结局。四种亚型是该规则下的变体模型，创作前先确定当期视频所属亚型。*

| 亚型 | 核心公式 | 情绪价值 | 主引擎 | 叙事焦点 |
|------|----------|----------|--------|---------|
| 🔴 **致命疏忽型** | 事故现场 + 被忽略的小人物 + 偶然的疏忽 | 倒吸凉气的后怕 | 因果震撼 | 偶然性、系统性漏洞 |
| 🟡 **清醒背叛型** | 文明冲突的十字路口 + 通晓两端的"夹缝人" + 清醒的生存抉择 | 人性的颤抖 | 道德撕裂 | 身份困境、生存的残酷 |
| 🟢 **徒劳先知型** | 集体狂热的浪潮 + 唯一清醒的反对者 + 被无视的正确预言 | "众人皆醉我独醒"的无力感 | 悲壮坚持 | 理想主义、被孤立的勇气 |
| 🔵 **荒诞工具型** | 荒诞的历史闹剧 + 认真负责的执行者 + 被系统背叛的结局 | 系统如何吞噬个体的荒诞感 | 黑色反差 | 体制的荒谬、个体被吞噬 |

**辅引擎**：
- **脆弱同盟**：展现各方势力（家族、君臣、盟友、同僚）关系的脆弱与算计。亚型选"徒劳先知型"时可弱化（矛盾常是单向的）；其余亚型正常使用。

### 风格铁律（创作前必阅）
1. **【3秒定生死法则】**：视频开头0-5秒，必须向**对该历史事件零认知的观众**，用画面+字幕/旁白，清晰交代：**时代、核心冲突双方、已不可逆转的态势**。这是流量的生死线。
2. **【观察眼绑定法则】**：严禁上帝视角说教。叙事应优先绑定在"局外人"主角的有限视角上，用"他看到…"、"他听见…"、"他注意到…"来推进情节。按亚型区分执行强度：
   - **致命疏忽型 / 荒诞工具型** → 严格执行，不得视角跳转
   - **清醒背叛型 / 徒劳先知型** → 可放宽（≤30%跳转），适度跳出有助于呈现"夹缝感"或"孤独感"
3. **【拒绝唯一决定论】**：严禁将宏大的历史结果简化为主角一人的瞬间抉择。必须通过台词或画面暗示其背后系统的崩溃。按亚型区分执行强度：
   - **致命疏忽型 / 荒诞工具型** → 严格执行，核心张力正在于"系统 > 个体"
   - **清醒背叛型 / 徒劳先知型** → 可放宽，叙事围绕主角的个人选择/坚持展开，过度强调系统原因会稀释困境感
4. **【工具人结局祛魅】**：主角的最终结局，必须点明其在权力结构中的**工具性本质**。他是"用过即弃的钥匙"、"承担骂名的幌子"、"测试忠诚的试纸"。避免简单的忠奸褒贬。
5. **【金句收尾点睛】**：视频最后3-5秒，必须用一句高度凝练、具有普遍共鸣的金句收尾，升华"局外人"的宿命，引发观众回味与传播。
6. **台词文学性铁律**：

   整体原则：**每句台词必须是"叙事链"中的一环**，不能是孤立的信息点。一段好的台词 = 场景锚点 + 感官细节 + 矛盾/因果。

   - **核心命题前置**：动笔前先定一句话核心命题（如"得手比失手更难"），和为这个命题量身定做的戏剧钩子（如"在跳舞时杀人"）。每段台词都在为呈现这个命题服务，不能跑题。
   - **比喻要具体**：比喻不是"像...一样"的空洞修辞，而是借用具体物象让人"看见"抽象概念。

     | | 写法 |
     |:---:|------|
     | ❌ | 他的内心充满了矛盾 |
     | ✅ | 他左手攥着提拔令，右手握着辞呈——两页纸一样重。 |
     | ❌ | 整个系统正在崩溃 |
     | ✅ | 警报灯一盏接一盏亮起来，像多米诺骨牌。 |

   - **金句不是堆词，是洞察压缩**：用最少的字，让观众"啊"一声。金句必须符合**可传播**特征——让观众愿意截图、转发。

     | | 写法 |
     |:---:|------|
     | ❌ | 这就是战争的残酷，没有谁对谁错。 | 空话，不痛不痒 |
     | ✅ | 他至死都不知道，自己只是那场面试的"试纸"。 | 有信息量、有反转 |
     | ✅ | 历史记住了那个按钮，却没记住按按钮的那只手。 | 具象、反常识 |

   - **句式节奏模板**：在关键分镜强制使用以下节奏切换。

     | 情绪 | 节奏 | 示例 |
     |------|------|------|
     | 紧张/悬念 | 短句堆叠，句号密集 | "他把信拆开了。读了三遍。然后脸色变得惨白。" |
     | 沉思/悲悯 | 一句长句，逗号撑起 | "在那些漫长的冬夜里，他反复想起那个决定——如果当时没有转身，命运会不会是另一副面孔。" |
     | 爆发/转折 | 短促有力的单字断句 | "他没有犹豫。三秒。四秒。然后——爆炸。" |

### 强制节奏模板（6段分镜）

视频须严格按以下六个情节-情绪段落依次展开。

#### 密度设计铁律

**仅有一条硬规则：不得连续两段 🔴。** 每段 🔴 后面必须跟一段 🟡 或 🟢，给观众时间消化。其余密度分配、无配音段的位置，完全由当期故事的叙事曲线决定。

**段落固有属性**（不可改动）：

| 段落 | 固有属性 |
|------|---------|
| 1.【暴击钩子】 | 必须有配音。开篇需要快速建立认知，旁白是最有效的工具 |
| 6.【主题回声】 | 必须有配音。金句需要声音承载，慢语速收束 |
| 其余4段 | 无限制。可根据故事节奏自由安排无配音/有配音 |

#### 密度设计原则（非规则，是参考思路）

在遵守"不得连续两段 🔴"的前提下，以下三种常见模式供选择，具体选用哪一种取决于故事的情绪曲线：

**模式A：外部压力型（压力来自外部环境，全程紧绷）**
```
1🔴 → 2🟡 → 3🔴 → 4🟡 → 5🔴 → 6🟢
```
→ 全程有配音，无无配音段。外部压力一刻不停，旁白贯穿始终，不给观众"落下来"的机会。适合节奏极快、冲突密集的故事。

**模式B：人物困境型（戏剧张力来自主角自身的选择）**
```
1🔴 → 2🟢(无配音) → 3🔴 → 4🟡 → 5🔴 → 6🟢
```
→ 无配音段放在第2段（局外人入场）。开头暴击后，让观众在沉默中"凝视"主角的尴尬处境——画面替旁白说话。适合主角身份本身就充满故事性的选题。

**模式C：情绪过山车型（节奏起伏最大，先扬后抑再扬）**
```
1🔴 → 2🟡 → 3🔴 → 4🟢(无配音) → 5🔴 → 6🟢
```
→ 无配音段放在第4段（内部撕裂）。前半程快速推进，到第4段突然收声，让观众在静默中感受撕裂感，为第5段的最终爆发蓄势。适合转折强烈、前后反差大的故事。

**密度与配音参数速查**：

| 密度 | 推荐配音参数 | 无配音替代方案 |
|:----:|:------------|:--------------|
| 🔴 高密 | speed=1.1~1.2, emotion=fluent/fearful, intensity=10~20 | （必须有配音） |
| 🟡 中密 | speed=0.95~1.05, emotion=calm, intensity=0 或省略 | （可选，如剧情需要可改为无配音） |
| 🟢 低密 | speed=0.8~0.9, emotion=calm/sad, intensity=-10~-30 | 无配音时设目标时长3-6秒，画面+环境音主导 |

> 核心思想：**无配音段的位置由故事决定，不是由模板决定。** 唯一硬约束是不得连续两段 🔴，在这个约束内，AI 根据故事的情绪曲线自由分配密度和配音模式。

### 选题偏好
- 必须是大历史事件的边缘参与者、执行者或意外关联者。
- 其结局最好具有悲剧性或讽刺性，或被主流历史刻意淡忘。
- 选题方向与对应的亚型匹配参考：
  - **致命疏忽型** → 寻找被无数人忽略但至关重要的细节，主角的疏忽须是"可理解的"（在当时看似合理）。
  - **清醒背叛型** → 寻找双重身份/双重忠诚之间的困境，没有"正确"答案，每个选择都有切肤的道德代价。
  - **徒劳先知型** → 寻找有历史依据的警告被时代局限性淹没的故事。主角的警告不能是迷信或谣言。
  - **荒诞工具型** → 不强调两难抉择，叙事张力来自"执行本身"——主角认真做一件本身就很荒诞的事。
- 避免已被过度神话、缺乏解构空间的历史顶流人物（如关羽、诸葛亮等）。
- 避免史料极端匮乏、主要依赖野史和民间传说的人物与事件。

## 制作规范
*本系列视频的标准化技术参数，确保统一的输出质量。*

| 参数 | 规范值 | 说明与备注 |
| :--- | :--- | :--- |
| **分镜段数** | 6-12段 | **这是成本控制的核心参数。** 每个分镜对应一次 AI 视频生成（固定成本），控制段数即控制成本。AI 按此上限规划，不可超纲。单分镜台词 ≤56字（≈16s配音时长），超出需拆为多个AI镜头。 |
| **台词总字数** | 200-450字 | 旁白驱动的叙事风格，按3.5字/秒语速，全文配音时长约57-129秒。 |
| **单分镜台词** | 17-56字 | 结合语速与拉伸上限，单次AI生成可覆盖 ≤56字。典型值30-40字/段。 |
| **单镜头生成限制** | 8秒 | AI 视频模型单次生成上限，非内容时长目标。剪辑时通过变速对齐实际配音时长。 |
| **单分镜拉伸上限** | 视场景运动类型而定 | `setpts = 配音时长 / 视频时长`。静态 ≤2.0x / 一般 ≤1.5x / 剧烈 ≤1.3x，超限卡顿。硬上限 ≤2.0x。 |
| **画面比例** | 9:16 竖版 | 适配手机短视频平台。 |
| **旁白** | 沉稳、冷静的男声 | 语速适中，富有叙事感，避免播音腔或过度煽情。 |
| **字幕** | 关键信息（人名、时间、冲突点）须用动态大字幕强化 | 确保在无声音环境下也能理解核心信息。 |
| **背景音乐** | 低沉、富有氛围感的原创或免版权音乐 | 情绪随叙事节奏起伏，但在旁白时段需自动减弱。 |
| **音效** | 关键动作（古代示例：开门、剑鸣、落笔；近现代示例：电报声、电话接通音、公文盖章声）须搭配有质感的音效 | 增强临场感与戏剧张力。 |
