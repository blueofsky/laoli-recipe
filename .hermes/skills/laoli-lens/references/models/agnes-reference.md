# Agnes 模型参考

## ⚠️ AI生图已知陷阱

以下为 agnes 模型在 laoli-lens 管线中的已知问题和应对策略：

| 问题 | 表现 | 应对 |
|------|------|------|
| **签名水印** | agnes 生成的图片右下角可能出现画师签名风格的水印，即使 prompt 包含 "no watermark" | 无法通过 prompt 消除。后期视频剪辑阶段通过裁剪/缩放弱化，或用 inpainting 去除。不影响叙事的轻微水印可接受。 |
| **种族面孔偏差** | 仅写 "Soviet" 时，模型可能生成东亚面孔 | prompt 中使用具体种族描述："Eastern European Slavic"、"Caucasian"，而非仅用国别。定妆照步骤务必验证面孔种族特征。 |
| **手部动作方向不可控** | prompt 描述"手从按钮上移开"，模型可能生成"手按在按钮上" | 手部动作是 AI 生图的弱项。如果关键叙事依赖手部动作方向，需多次重试并用 vision_analyze 验证。无法保证一次成功。 |
| **军衔/肩章不精确** | prompt 要求"中校2颗星"，模型可能生成3颗星或错误军衔 | 肩章星数属于小尺寸细节，AI 视觉模型计数可能有误差。定妆照生成后务必用 vision_analyze 放大确认。 |
| **年龄偏差** | prompt 要求"44岁"，模型可能生成50-60岁面孔 | 在 prompt 中同时强调"young for his rank"或具体年龄特征（如"no grey hair"），但效果有限。可接受±10岁偏差。 |
| **数量控制不精确** | prompt 要求"5条导弹轨迹"，模型可能生成4条 | AI生图难以精确控制元素数量。如果关键叙事依赖精确数量（如台词"五枚"），需用 vision_analyze 验证。差异1-2个可接受（画面快速切换时观众不会细数）。 |

## 🔧 Agnes API 限制

- **RPM 限制**：每分钟 20 次请求（RPM20），超过后返回 503 ServiceUnavailableError
- **应对策略**：批量生成时控制并发数（`--jobs 2` 而非默认 4）；连续生成间隔至少 3 秒；遇到 503 错误时 sleep 30-60 秒后重试
- **替代方案**：nano banana pro 在某些场景下表现优于 agnes（更精确的数量控制、更好的种族面孔还原），可作为备选 provider
