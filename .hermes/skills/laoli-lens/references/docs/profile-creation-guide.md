# Profile 创建指南

## Profile 文件结构

Profile 文件位于 `references/profiles/<方案名>.md`，必须包含以下4个章节：

```markdown
# 方案配置 - [方案名]

## 方案设定
（元信息、核心理念、受众定位）

## 视觉风格设定
（风格标签、视觉锚点、构图、光影、时代适应性）

## 叙事设定
（叙事人设、叙事类型/亚型、风格铁律、强制节奏模板、选题偏好）

## 制作规范
（分镜段数、台词字数、画面比例、旁白、字幕、BGM、音效等技术参数）
```

## 各章节被步骤引用的情况

| 章节 | 被哪些步骤引用 |
|------|--------------|
| 方案设定 | 第1步（选题大纲） |
| 视觉风格设定 | 第2步（视觉执行指令）、第4步（定妆照） |
| 叙事设定 | 第1步（选题大纲）、第3步（分镜脚本） |
| 制作规范 | 第3步（分镜脚本） |

## 创建新Profile的要点

1. **方案设定**要明确定位和受众，避免与现有方案重叠
2. **视觉风格设定**必须提供可直接嵌入IMAGE PROMPT的「风格后缀」
3. **叙事设定**中的「叙事类型/亚型」表必须定义核心公式、情绪价值、叙事焦点
4. **风格铁律**必须包含可执行的规则（如"3秒钩子法则"、"倒叙优先法则"）
5. **强制节奏模板**必须定义6段分镜的结构和密度设计铁律
6. **制作规范**的参数必须与步骤文件中的约束一致
