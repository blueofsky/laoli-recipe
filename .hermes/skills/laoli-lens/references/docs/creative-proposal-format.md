# 创意方案格式

> 本文档定义 laoli-lens skill 第1步的输入格式。创意方案是**用户提供的原始素材**，skill 会读取后自动生成选题大纲、视觉指令、分镜脚本等产出物。

## 格式说明

创意方案只需提供"故事是什么"——历史现场、人物、事件、影响、张力。不需要包含3秒钩子、节奏段划分、叙事引擎等——这些由 skill 根据方案配置自动生成。

## 必填字段

```markdown
# [选题名]

> 方案：[方案名]
> 叙事类型：[从方案配置的叙事类型表中选择]

---

- **历史现场**：[具体时间、地点、时代背景]
- **核心人物**：[姓名、身份、年龄、关键特征]
- **关键事件**：[发生了什么，因果链是什么]
- **巨大影响**：[这件事导致了什么后果]
- **启示性结果**：[一句话总结这个故事的核心洞察]
- **故事张力**：[这个故事的戏剧冲突在哪里]
- **这个故事为什么"有意思"？**：
  1. **颠覆认知**：[观众以为X，但99%的人不知道Y]
  2. **极强的代入感**：[如果你是主角，你会怎么选]
  3. **深刻的现代隐喻**：[这个故事像不像当下的某件事]
- **情绪价值**：[观众看完会有什么感受]
- **现代隐喻**：[这个故事映射到当下是什么]
```

## 参考示例

见项目目录 `文档/选题库/` 下的选题文件：
- `彼得罗夫事件.md` — 灾难中的普通人
- `协和号空难.md` — 蝴蝶效应
- `特内里费空难.md` — 灾难中的普通人

## 常见错误

❌ **错误**：在创意方案中包含分镜脚本、3秒钩子、节奏段划分等产出物
→ 这些是 skill 的工作，不是用户的输入

❌ **错误**：创意方案过于简略（只有一句话梗概）
→ skill 需要足够的故事素材才能生成高质量产出

✅ **正确**：创意方案只提供故事素材，让 skill 独立完成创作加工
