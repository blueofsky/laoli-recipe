# 创意方案格式

> 本文档定义阶段A（创意研发）的产出物格式。创意方案是**阶段A的产出**，包含选题诊断、多角度切入、精修方案、视觉策略、金句候选等创意要素。后续技术执行阶段（第1-12步）读取此文件自动生成创作底本。

## 与旧流程的区别

| | 旧流程 | 新流程（阶段A） |
|------|--------|----------------|
| 谁写创意方案 | 用户手写，作为skill输入 | 阶段A自动产出，用户只需选题+选方向 |
| 内容深度 | 只有故事素材 | 包含诊断、多角度、视觉策略、金句 |
| 格式 | 自由格式 | 结构化格式（见下方） |
| 迭代次数 | 无 | 至少3个角度对比，用户选最优 |

## 产出格式

```markdown
# [项目名]

> 方案：<方案名>
> 叙事类型：<类型名>
> 创意角度：<角度名称>

## 诊断
- 视觉爆点：[这个故事的命运转折点物件是什么]
- 意外感：[观众会"啊？就因为这个？"吗]
- 辨识度：[跟其他AI历史视频区分得开吗]

## 钩子
| 版本 | 台词 | 画面 | 适用场景 |
|------|------|------|---------|
| V1 | ... | ... | 首选 |
| V2 | ... | ... | 备选 |
| V3 | ... | ... | 备选 |

## 叙事结构
**结构名称**：[如"物件聚焦式""倒叙剥洋葱""双线交叉"]

| 段落 | 时长 | 内容 | 情绪 | 视觉策略 |
|------|------|------|------|---------|
| 1 | 0-5s | ... | 震撼 | ... |
| 2 | 5-15s | ... | 代入 | ... |

## 视觉记忆点
- 核心：[观众看完会记住的那个画面]
- 辅助：[2-3个次要视觉元素]
- 禁忌：[本选题要避免的视觉元素]

## 金句候选
| 金句 | 类型 | 适用位置 |
|------|------|---------|
| ... | 刺痛型/洞察型/反差型 | 结尾/中段转折 |

## 标题候选
1. [悬念型标题]
2. [冲击型标题]
3. [含蓄型标题]

## 选中的标题
**[用户确认的最终标题]**
```

## 参考示例

见项目目录 `文档/选题库/` 下的选题文件（旧格式，仅供参考故事素材）：
- `彼得罗夫事件.md` — 灾难中的普通人
- `协和号空难.md` — 蝴蝶效应
- `特内里费空难.md` — 灾难中的普通人

> ⚠️ 选题库中的文件是"故事素材"，不是"创意方案"。创意方案由阶段A自动产出，包含更丰富的创意要素。
