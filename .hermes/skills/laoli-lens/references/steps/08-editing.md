# 第8步：视频剪辑

处理分镜视频：先对齐配音时长，再按需检查并去水印，然后提取音效，输出处理报告。所有操作均使用 `ffmpeg` 完成。

**输入**：分镜视频（`素材/视频/原始/`，原始文件不动）+ 配音（`素材/配音/`）

### 8a. 尺寸校正（agnes 视频）

agnes 视频 API 固定输出 704 宽，无视 `--size` 参数，导致画面不是标准 9:16。需要裁剪到正确比例。

- **检测**：对每个原始视频执行 `ffprobe -v error -select_streams v:0 -show_entries stream=width,height -of csv=p=0`
- **判断**：如果宽高比 ≠ 9:16（允许 ±0.005 误差），则执行裁剪
- **裁剪逻辑**：
  - 目标：宽 = 高 × 9 ÷ 16，四舍五入到偶数
  - 水平居中裁剪：`crop={目标宽}:{高}:({原宽}-{目标宽})/2:0`
  - 常见情况：
    - 704×1152 → crop=648:1152:28:0
    - 704×1280 → crop=720:1280:-8:0（即左侧裁8px）
  - 如果已经是标准 9:16，跳过
- **命令**：`ffmpeg -i "素材/视频/原始/scene0x_描述.mp4" -vf "crop={目标宽}:{高}:{x偏移}:0" -c:v libx264 -crf 18 -an "素材/视频/原始/scene0x_描述.mp4"`（覆盖原文件，原始已不需要保留）
- **输出**：覆盖写回 `素材/视频/原始/scene0x_描述.mp4`

> ⚠️ agnes 图片 API 尺寸正常，无需校正。仅视频需要。

### 8b. 变速对齐

将视频时长调整至与配音时长一致。
- **输入**：`素材/视频/原始/scene0x_描述.mp4` + `素材/配音/scene0x_描述.mp3`
- **系数计算**：`setpts = 配音时长 / 视频时长`
- **处理方式**：纯变速（`setpts`），不加补帧，去掉音频轨道（`-an`）
- **输出**：`素材/视频/变速/scene0x_描述.mp4`（无声）

### 8c. 去水印（按需执行）

> 如果分镜视频全部来自 AI 生成，则没有水印，直接跳过本步。

检查 `素材/视频/变速/` 中的视频是否有平台水印（如 Veo），仅在确认有水印时执行裁剪。

- **有水印时**：底部裁剪约 70px，精确 9:16（`crop=1040:1850:20:0`），输出到 `素材/视频/精修/`
- **无水印时**：跳过本步，剪映直接使用 `变速/` 中的文件

### 8d. 提取音效

从原始视频中提取音频，先保存原始音效，再按配音长度变速得到成品音效。

**第1步——提取原始音频**：
- **命令**：`ffmpeg -i "素材/视频/原始/scene0x_描述.mp4" -vn -c:a libmp3lame -q:a 2 "素材/音效/原始/scene0x_描述.mp3"`
- **输出**：`素材/音效/原始/scene0x_描述.mp3`（原始音频文件，保留备份）

**第2步——变速对齐配音**：
- **系数计算**：`atempo = 原始时长 / 配音时长`
  - ⚠️ 与 8a 的 `setpts` 互为倒数，不要直接复用公式
  - atempo 仅支持 0.5~2.0 范围，超出需链式调用
- **命令示例**：`ffmpeg -i "素材/音效/原始/scene0x_描述.mp3" -vn -c:a libmp3lame -q:a 2 -filter:a "atempo={系数}" "素材/音效/变速/scene0x_描述.mp3"`
- **输出**：`素材/音效/变速/scene0x_描述.mp3`
