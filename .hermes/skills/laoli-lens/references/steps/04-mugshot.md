# 第4步：生成角色定妆照

**调用**：`laoli-image` skill

- **输入**：`核心角色设定`中的「定妆照提示词」
- **输出**：竖屏9:16图片，保存到 `素材/定妆/ref_角色ID.jpg`
- **数量**：每个核心角色1张
- **生成顺序**：按`核心角色设定`的角色顺序逐个生成

> **不需要定妆照的角色**：群众/配角（无角色设定）

**批量模式说明**：多个角色时 agent 可能会走 `laoli-image` 的 batch 模式。请将 `batch.json` 生成到 `素材/定妆/` 目录下（而非项目根目录），使得 `image`、`ref` 等路径自然以 `素材/定妆/` 为基准解析。

### ⚠️ 生成后必须验证

定妆照生成后，**必须用 `vision_analyze` 逐项核实**以下角色设定要素，不匹配则调整 prompt 重新生成：

| 验证项 | 检查方法 | 常见偏差 |
|--------|---------|---------|
| **种族/民族** | 观察面部轮廓、眼窝深度、鼻梁高度、肤色 | AI 默认可能生成东亚面孔，需在 prompt 开头强调 "Eastern European"、"Slavic"、"Russian" 等种族特征 |
| 年龄 | 视觉判断面部状态 | AI 倾向生成偏老5-10岁的面孔 |
| 军衔/身份标识 | 放大观察肩章、徽章等细节 | 星数容易少或多，军衔级别易混淆 |
| 勋章/装饰 | 确认是否有多余勋章 | AI 倾向添加勋章装饰 |
| 体型 | 观察肩宽和面部瘦削程度 | AI 倾向生成魁梧体型 |
| 气质 | 整体神情和姿态 | 威严 vs 疲惫需要在 prompt 中明确 |

### ⚠️ AI水印瑕疵

AI生图模型（特别是agnes）可能在图片右下角添加画师签名水印，即使prompt中包含 "no text, no letters, no watermark"。这是模型固有瑕疵，如果水印较小且在暗色背景下不明显，可以接受，不必反复重试。

### 📝 迭代生成策略

首次生成几乎不可能完全匹配角色设定。迭代时：

1. **简化 prompt**：去除过于详细的风格描述，聚焦于年龄、体型、气质、身份标识等核心差异项
2. **用否定排除**：明确写出 `no medals, no decorations` 等排除不需要的元素
3. **关键词前置**：将 `thin`、`tired`、`44-year-old` 等关键特征放在 prompt 开头
4. **每次只改1-2项**：避免一次改太多导致其他项也变化
