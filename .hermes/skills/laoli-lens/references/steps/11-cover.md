# 第11步：生成封面图

基于选题核心视觉意象设计并生成视频封面图。

**输入**：选题大纲（选题标题/核心冲突/视觉符号）

**设计要点**：
- 封面必须预留标题文字空间（通常上方1/3做深色渐变或留黑）
- 画面聚焦选题核心视觉符号（如电报桌、耳机、摩尔斯纸带、煤油灯等）
- 风格与视频统一，保持方案配置的视觉风格
- 竖版3:4比例（视频号封面标准比例）

**调用**：`laoli-image` skill（单图模式，不走 batch）

**输出**：`素材/封面/项目名.jpg`

---
## ✅ 本步完成条件

封面图生成完毕后，必须通过 Task tool 调用 `laoli-lens-reviewer` 进行评审（默认 L2：最终视觉产出检查）。仅当评审返回 **✅ 通过** 后，方可告知 Owner 本步完成、进入下一步。**未完成评审 = 本步未完成。**
