# A0-创意研发 评审检查清单

## 默认级别：🟡 L2（标准评审）

> 阶段A评审**创意潜力**而非格式合规。格式是次要的，关键是方案有没有"让人想停下来听"的能力。

### 钩子冲击力（30%）
- [ ] 开场钩子能在3秒内让人停下来
- [ ] 钩子展示了最震撼的画面/台词（不是交代背景）
- [ ] 提供了3个钩子版本，有对比
- [ ] 钩子可视觉化（具体画面，非抽象概念）

### 意外感（25%）
- [ ] 因果链有反转（不是"坏人做坏事"）
- [ ] 观众会"啊？就因为这个？"
- [ ] 选题符合方案配置的选题偏好（意外感+小人物代入+自然隐喻）

### 视觉辨识度（20%）
- [ ] 有明确的"视觉记忆点"（观众会记住哪个画面）
- [ ] 跟其他AI历史视频有区分度
- [ ] 视觉策略避免了场景单一问题
- [ ] 有"命运转折点物件"的视觉锚点

### 金句质量（15%）
- [ ] 金句是"刺痛"不是"说教"
- [ ] 金句从故事中自然生长（不是外挂的总结）
- [ ] 提供了3个金句候选，有类型区分
- [ ] 无"机器没有良心但人有"类口号式金句

### 风格铁律合规（10%）
- [ ] 符合3秒钩子法则
- [ ] 符合拒绝说教法则
- [ ] 符合观察眼绑定法则
- [ ] 符合金句不是口号法则
- [ ] 如覆盖倒叙优先法则，已在创意方案中说明理由

### 格式完整性
- [ ] 产出文件存在且路径正确（创意方案.md）
- [ ] 必填字段完整：诊断、钩子、叙事结构、视觉记忆点、金句候选、标题候选
- [ ] 叙事结构段落表有明确的时长、情绪、视觉策略
- [ ] 至少有3个标题候选
