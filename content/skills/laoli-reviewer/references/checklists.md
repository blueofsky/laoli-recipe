# 检查灵感库

> Reviewer 评审时的参考索引，非强制清单。按资产类型 + 级别组织，Reviewer 根据实际情况灵活选用。

---

## 📄 文案类（L1 基础 / L2 升级）

适用：创作底本、配乐方案、发布文稿、字幕文稿

**L1**：确认文件存在且非空、编码正常  
**L2 升级检查**（涉及数据/引用时）：
- 内部链接/引用指向存在的文件
- 数据数字与上游一致（分镜数、时长等）
- 台词逐段匹配原始脚本（字幕文稿）

### 类型关注点

| 文档类型 | 特别关注 |
|---------|---------|
| 创作底本 | 分镜完整性、视觉符号一致性、台词与情绪匹配度 |
| 字幕文稿 | 与原始脚本逐字对比、标点处理、金句处理 |
| 配乐方案 | 情绪分段时长与实际音频匹配、关键词覆盖面 |
| 发布文稿 | 标题吸引力、钩子质量、标签相关性 |

---

## 🖼️ 图片类（L2 标准）

适用：定妆照、分镜图、封面图

**L1**：文件存在、数量正确、大小合理、格式正确  
**L2**：目视抽查关键帧、分辨率在预期范围、核心元素与 Prompt 匹配、角色一致性

### 类型关注点

| 图片类型 | 特别关注 |
|---------|---------|
| 定妆照 | 角色特征清晰度、年龄/表情符合设定、无多余文字/水印 |
| 分镜图 | 构图与脚本描述匹配、视觉符号 Callback、风格统一 |
| 封面图 | 标题区域留白充足、主体突出、适合目标平台比例 |

### 常见问题速查

```
🔴 致命：完全错误画面 / 图片损坏 / 明显水印/Logo/品牌名
🟠 严重：面部畸变/多手指 / 文字乱码 / 色调风格严重不一致
🟡 一般：细节瑕疵 / 构图偏移 / 光影方向不一致
```

---

## 🎬 视频类（L2 标准 / L3 深度）

适用：生成视频、视频剪辑

**L1**：文件存在、数量正确、时长>0、格式正确  
**L2**：抽取关键帧目视检查、分辨率符合预期、时长合理、内容与分镜/Prompt 匹配  
**L3 额外**（视频剪辑步骤）：
- ffprobe 全量提取时长
- 音视频时长误差<阈值
- 变速系数<3.0
- 裁剪后分辨率记录
- 无水印帧抽查
- **音效文件完整（每个分镜对应一条）**
- **音效变速后时长≈配音时长**

### 步骤关注点

| 步骤 | 特别关注 |
|------|---------|
| 生成视频 | 运动幅度合理、循环感、画质 |
| 视频剪辑 | 音画对齐精度、变速后流畅度、裁剪完整性、音效文件完整且时长匹配 |

---

## 🔊 音频类（L3 深度验证）

适用：配音

**L1**：文件存在、数量正确、时长>0、格式正确  
**L2**：波形特征检查、文件大小与时长成比例  
**L3**：ffprobe 全量提取（codec/sample_rate/channels/bit_rate/duration）→ 参数一致性（采样率/声道数/比特率统一）→ 时长合理性（单段 vs 字数 ≈ 3-4字/秒）→ 特殊处理验证（金句处理、标点处理、TTS 参数一致性）

### 常见问题速查

```
🔴 致命：TTS 读错台词 / 大段静音或杂音 / 文件无法播放
🟠 严重：时长严重偏离 / 参数不一致 / 音色语调突变
🟡 一般：语速偏快偏慢 / 个别发音不准 / 音量不均匀
```

## AI 生成已知陷阱

Reviewer 检查时需要主动关注的常见 AI 生成缺陷，按生成技术分类。

### 图片类（文生图 / 图生图 / 参考图加权）

| 陷阱场景 | 表现 | 检查要点 |
|---------|------|---------|
| 面部畸变 | 五官扭曲、眼球变形、手指数量异常 | 目视检查人物面部，特别关注眼睛、手指、牙齿 |
| 文字乱码/多余文字 | 画面中出现无法辨认的字符或品牌名 | 扫描画面所有文字区域，尤其是背景中的招牌、海报 |
| 角色克隆 | 多角色使用同一参考图时特征串混 | 对比每个定妆照的角色特征，检查是否出现特征交叉 |
| 参考图权重过高 | 构图被参考图锁死，无法适配新 prompt | 检查生成结果是否显著偏离 prompt 描述的核心元素 |
| 风格不一致 | 同一角色在不同分镜中的风格/比例不统一 | 跨帧对比人物面部、服装细节、光影方向 |
| 多余水印/Logo | 生成图片中携带模型平台水印 | 目视检查画面边缘和角落 |

### 视频类（图生视频 / 文生视频）

| 陷阱场景 | 表现 | 检查要点 |
|---------|------|---------|
| 运动幅度过大 | 画面主体抖动、镜头摇晃 | 播放检查关键帧过渡是否平滑，关注主体轮廓稳定性 |
| 画面闪烁 | 帧间不一致，出现闪烁或跳帧 | 连续播放多个分镜，检查视觉连续性 |
| 循环感/静止感 | 首尾帧相似，像在原地踏步 | 对比视频首尾帧，检查是否有实际运动变化 |
| 主体变形/消失 | 视频中角色或关键物体变形、消失又重新出现 | 全程跟踪主体轮廓，检查形状是否保持一致 |
| 运动僵化 | 人物动作不自然，有机械感 | 关注人体关节运动是否协调，表情变化是否自然 |

### TTS / 配音

| 陷阱场景 | 表现 | 检查要点 |
|---------|------|---------|
| 金句标记遗漏 | ★ 金句被配音读出（应只做字幕） | 逐段检查台词中金句标记（★/☆）是否被正确处理 |
| 标点渲染错误 | 省略号被读成"点点点"、引号影响语调 | 特别注意中文标点的发音处理是否正确 |
| 语调平 | 整段台词缺乏情绪起伏，读稿感强 | 抽查情绪高潮段落，检查语调是否匹配台词内容 |
| 生僻字发音错误 | 多音字/生僻字读错 | 检查台词中的易错字、专业名词、人名地名 |
| 语速异常 | 整体或局部语速明显偏快/偏慢 | 针对部分段落计时，参考正常范围 3-5 字/秒 |

### ffmpeg 处理

| 陷阱场景 | 表现 | 检查要点 |
|---------|------|---------|
| 变速过猛 | setpts > 3.0 导致人眼可感知不自然 | 检查变速系数是否超过安全范围（警戒值 2.0-3.0） |
| 音频变调 | atempo 超出 0.5-2.0 范围需链式处理但未处理 | 检查 atempo 参数设置及链式处理配置 |
| 裁剪偏移 | 裁剪后画面内容不完整或偏移 | 检查裁剪后视频的分辨率和可视内容完整性 |
| 音画不同步 | 音频与视频时长不匹配 | 对比音频时长与视频时长，误差应 < 0.5s |
| 音效文件缺失 | 分镜数与实际音效文件数不匹配 | 检查音效文件数量是否等于分镜数 |

## 交叉验证矩阵

多步骤间依赖验证：

| 上游 | 下游 | 验证内容 |
|------|------|---------|
| 创作底本(台词) | 字幕文稿（台词提取） | 逐字匹配，无遗漏无多余 |
| 创作底本(分镜) | 分镜图 | 每个分镜都有对应图片 |
| 创作底本(Prompt) | 生成视频 | 视频内容与 Prompt 核心词匹配 |
| 字幕文稿（台词提取） | 配音 | 配音内容 = 台词内容（特殊处理后） |
| 配音(时长) | 视频剪辑(sync) | sync 视频时长 ≈ 音频时长 |
| 生成视频(有水印) | 视频剪辑(crop) | crop 后水印消失，时长不变 |
| 原始视频(音频) | 视频剪辑(音效) | 音效文件数 = 分镜数，每条音效变速后时长≈对应配音时长 |
| 分镜图(角色) | 定妆照 | 同一角色外观一致 |
| 发布文稿(标题) | 封面图(文字) | 主标题文字一致 |

## 边界值参考

判断"是否异常"的经验参考：

| 检查项 | 正常范围 | 警戒值 | 异常值 |
|--------|---------|:-----:|:-----:|
| 中文 TTS 语速 | 3-5字/秒 | <2 或 >6 | <1 或 >8 |
| 视频变速系数 | 0.5 - 2.0 | 2.0 - 3.0 | >3.0 |
| 音频采样率 | 16000/22050/32000/44100 Hz | 其他标准值 | 非标准值或混合 |
| MP3 比特率 | 64-320 kbps | <64 | >320 或 0 |
| 图片分辨率(竖版) | 1080x1920 左右 | <720 或 >4K | 过小或过大 |
| 视频单段时长 | 按项目规定 | 偏离±50% | 偏离±100%+ |
| 音视频时长误差 | <0.1s | 0.1-0.5s | >0.5s |

> 通用参考值，具体以项目要求为准。
