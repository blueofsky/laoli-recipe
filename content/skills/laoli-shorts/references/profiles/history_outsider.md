# 方案配置 - 历史局外人

## 方案设定
*本方案专为“历史的局外人”短视频系列设计，聚焦宏大历史叙事中被忽略、被裹挟的关键小人物，通过他们的瞬间抉择，折射时代巨变与人性的复杂光谱。*

- **核心调性**：深沉、悲悯、充满命运感。拒绝脸谱化的英雄或恶人叙事，追求在历史褶皱中发现人性的颤抖与微光。
- **价值主张**：历史由胜利者书写，但由无数“局外人”的代价构成。我们关注代价，理解复杂，从而更透彻地看清权力与人性的游戏规则。
- **受众画像**：20-40岁，对历史、权谋、人性故事有浓厚兴趣，不满足于史实复述，渴望获得情感共鸣与认知颠覆的泛知识群体。
- **一句话定位**：给普通人的权力游戏说明书。

## 视觉风格设定
*本系列追求电影级的历史厚重感与戏剧张力，视觉服务于叙事，营造沉浸与压迫并存的观看体验。*

- **核心视觉标签**：
  - **艺术风格**：Cinematic, realistic oil painting style, heavy historical texture
  - **画质与光影**：8K resolution, ultra-detailed, Rembrandt lighting with strong chiaroscuro (伦勃朗式光影)， cinematic film grain
  - **色彩**：整体偏向中低饱和度，突出棕、黑、金等厚重色调，避免现代数码感的鲜艳。
- **视觉锚点与构图**：
  1.  **主角特写**：必须强化面部特写，尤其是眼部，以传达恐惧、犹豫、决绝或空洞。面部是情绪的战场。
  2.  **环境对比**：利用光影强烈区分“权力中心”（光亮处）与“局外人徘徊之地”（阴影处）。
  3.  **关键符号**：每期视频须有一个核心的、可操作的“历史小物件”特写（一道密旨、一把钥匙、一杯毒酒、一块玉玦），作为命运的物理象征。
  4.  **构图**：多使用框架式构图、不对称构图，营造压抑或动荡感。定妆照采用半身像、深色纯背景，突出人物状态。
- **动态运镜**：
  - 以缓慢的推、拉、摇为主，模仿“凝视”与“审视”，营造沉浸感和时间流逝感。
  - 关键动作（如开门、举剑、落笔）可辅以细微的镜头震动或速度变化，强化冲击力。
  - 禁止无意义的快节奏闪烁剪辑。

## 叙事设定
*这是本系列的灵魂。所有创作必须严格遵循以下“铁律”，它们确保了内容的颠覆性、可看性与思想深度。*

- **叙事人设**：
  - 叙述者是一名冷静、抽离但心怀悲悯的“历史观察者”。语调沉稳，不煽情，但字句充满力量。
  - 与观众的关系是“分享洞察的老友”，而非“灌输知识的老师”。

- **五条风格铁律（创作前必阅）**：
  1.  **【3秒定生死法则】**：视频开头0-5秒，必须向**对该历史事件零认知的观众**，用画面+字幕/旁白，清晰交代：**时代、核心冲突双方、已不可逆转的态势**。这是流量的生死线。
  2.  **【观察眼绑定法则】**：严禁上帝视角说教。全部叙事必须严格绑定在“局外人”主角的有限视角上。用“他看到…”、“他听见…”、“他注意到…”来推进情节，让观众通过主角的眼睛亲自发现真相。
  3.  **【拒绝唯一决定论】**：严禁将宏大的历史结果简化为主角一人的瞬间抉择。必须通过台词或画面暗示其背后系统的崩溃（众叛亲离、大势已去）。主角是“关键一环”、“最后的稻草”，而非“唯一原因”。
  4.  **【工具人结局祛魅】**：主角的最终结局，必须点明其在权力结构中的**工具性本质**。他是“用过即弃的钥匙”、“承担骂名的幌子”、“测试忠诚的试纸”。避免简单的忠奸褒贬。
  5.  **【金句收尾点睛】**：视频最后3-5秒，必须用一句高度凝练、具有普遍共鸣的金句收尾，升华“局外人”的宿命，引发观众回味与传播。

- **叙事引擎与节奏模板（60秒标准结构）**：
  - **主引擎**：**生死抉择**。聚焦主角在个人/家族存亡与更高道义（忠君、理想、伦理）间的撕裂。
  - **辅引擎**：**脆弱同盟**。展现主角所处环境中，各方势力（家族、君臣、盟友）关系的脆弱与算计。
  - **强制节奏模板（6段分镜）**：
    视频须严格按以下六个情节-情绪段落依次展开。每个段落的预设时长供参考，但可根据叙事密度在分镜阶段灵活调整分镜数量。
    1. 【暴击钩子】（约0-5秒）：必须用强画面和台词，向零认知观众交代清楚时代、核心冲突双方与不可逆转的态势。
    2. 【局外人入场】（约6-12秒）：必须点明“局外人”主角的尴尬身份，以及他接收到的那个“不可能完成”或“充满矛盾”的任务。
    3. 【观察眼·外部压力】（约13-30秒）：必须严格通过主角的“观察眼”，呈现来自外部环境、权威或敌人的巨大压力与威胁。这是困境的建立。
    4. 【观察眼·内部撕裂】（约31-45秒）：必须继续通过主角的“观察眼”，展现系统内部的背叛、算计、沉默或情感拉扯。这是抉择的催化剂。
    5. 【沉默的代价】（约46-55秒）：必须展现主角行动（或不行动）的直接后果，并点明其在权力结构中的“工具”本质。
    6. 【主题回声】（约56-60秒）：必须用一句高度凝练的金句收尾，升华“局外人”的普遍宿命。

- **选题偏好**：
  - 必须是大历史事件的边缘参与者、执行者或意外关联者。
  - 其抉择必须在史料框架内有推理空间，且在道德或逻辑上充满两难。
  - 其结局最好具有悲剧性或讽刺性，或被主流历史刻意淡忘。

- **选题禁忌**：
  - 避免已被过度神话、缺乏解构空间的历史顶流人物（如关羽、诸葛亮等）。
  - 避免史料极端匮乏、主要依赖野史和民间传说的人物与事件。

## 制作规范
*本系列视频的标准化技术参数，确保统一的输出质量。*

| 参数 | 规范值 | 说明与备注 |
| :--- | :--- | :--- |
| **视频时长** | 58-62秒 | 严格围绕60秒结构展开，可微调。 |
| **分镜段数** | 6段以上 | 由管线根据节奏模板和单镜头生成限制自动规划 |
| **单镜头生成限制** | 8秒 | 默认值：8s，可根据模型的实际情况调整或用户自定义 |
| **画面比例** | 9:16 竖版 | 适配手机短视频平台。 |
| **旁白** | 沉稳、冷静的男声 | 语速适中，富有叙事感，避免播音腔或过度煽情。 |
| **字幕** | 关键信息（人名、时间、冲突点）须用动态大字幕强化 | 确保在无声音环境下也能理解核心信息。 |
| **背景音乐** | 低沉、富有氛围感的原创或免版权音乐 | 情绪随叙事节奏起伏，但在旁白时段需自动减弱。 |
| **音效** | 关键动作（开门、剑鸣、落笔）须搭配有质感的音效 | 增强临场感与戏剧张力。 |
