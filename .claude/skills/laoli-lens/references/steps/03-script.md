# 第3步：编写分镜脚本

**输入**：
- 选题大纲中的【节奏段】列表、3秒钩子
- 核心角色设定（视觉锚点写入 IMAGE PROMPT）
- 通用视觉准则（风格后缀追加到 IMAGE PROMPT）

### 智能分镜规划

1. **产出**：选题大纲中的"节奏段"列表 + 核心角色设定 + 通用视觉准则
2. **核心任务**：为每一个"节奏段"创作分镜脚本。节奏段与分镜不是1对1关系。
   - **拆分规划**：根据情节密度、情绪转折点，将每个节奏段拆为1个或多个分镜
   - **命名格式**：`## 分镜 N / [节奏段名]-[镜头描述]`
   - **分镜数约束**：全部分镜总数控制在方案定义的「分镜段数」范围内
   - **时长弹性**：单个分镜 AI 生成时长不超过方案「单镜头生成限制」
   - **台词上限**：单分镜台词量按方案「单分镜台词」上限控制
   - **叙事连贯**：跨分镜台词需保持语言和情绪的连贯
3. **输出**：每个分镜独立输出"台词"+"角色参考图"+"IMAGE PROMPT"+"VIDEO MOTION"

### 3a-0. 节奏密度规划

在写台词之前，先为所有分镜分配以下三组参数，决定每个分镜的叙事密度和配音策略。

**① 密度等级**（控制观众的信息接收压力）：

| 密度 | 含义 | 配音策略 | 推荐参数 |
|------|------|---------|---------|
| 🔴 高密 | 信息密集，叙事快速推进 | 有配音 | speed=1.1~1.2, emotion=fluent/fearful, intensity=10~20 |
| 🟡 中密 | 信息适中，节奏从容 | 有配音 | speed=0.95~1.05, emotion=calm, intensity=0或省略 |
| 🟢 低密 | 画面主导，观众消化时间 | 无配音(画面+narrator沉默) 或 有配音(speed放慢) | speed=0.8~0.9, emotion=calm/sad, intensity=-10~-30 或 无配音 |

**核心铁律：不得连续出现两段 🔴 高密。** 每段 🔴 后面必须跟一段 🟢 或 🟡，给观众留出"感受画面"的时间。

**② 配音模式**（决定是否生成配音文件）：

- **有配音**：该分镜生成配音音频。配音文本中可嵌入 MiniMax 行内气息标签达成自然的气口
- **无配音**：该分镜不生成配音音频，由画面+环境音+BGM主导。**无配音段就是观众的气口**

**③ 配音指令（仅配音模式=有配音时需标注）**：

| 参数 | 范围 | 说明 | 对应 laoli-tts 参数 |
|------|------|------|---------|
| speed | 0.8~1.2 | 🔴高密段1.1~1.2，🟡中密段0.95~1.05，🟢低密段0.8~0.9 | `--speed` |
| emotion | calm/fluent/sad/fearful | calm=沉稳叙述 fluent=流畅推进 sad=悲悯下沉 fearful=紧张恐惧 | `--emotion` |
| intensity | -30~30（推荐）<br>API支持-100~100 | 🔴高潮段=10~30(有力)，🟢下沉段=-30~-10(轻柔)，🟡中性=0或省略 | `--intensity` |

在创作底本中输出密度规划表：

```markdown
**密度规划**：
| 分镜 | 密度 | 配音模式 | speed | emotion | intensity |
|:----:|:----:|:--------:|:----:|:-------:|:---------:|
| 1 | 🔴 | 有配音 | 1.15 | fluent | 20 |
| 2 | 🟢 | 无配音 | — | — | — |
| 3 | 🟡 | 有配音 | 0.9 | calm | -15 |
| ... |
```

### 3a. 台词

- 遵循方案「制作规范」或「风格铁律」中对台词的特别要求
- 每段2-4句话，**总字数严格遵循方案配置的台词总字数上限（单分镜≤上限值）**，字数到位但绝不超限。**不得敷衍写成1句话**，每段台词必须包含：场景交代 + 人物状态 + 矛盾/转折
- **质量红线**：agnes模型支持3-15秒视频生成，可根据台词时长灵活选择生成时长。若配音时长与视频时长不匹配，可用变速调整，但尽量让生成时长匹配配音时长
- 每段台词必须包含感官细节（视觉/听觉/触觉至少选其一），禁止写"他感到...""他认为..."这类抽象陈述
- 金句位置用 `★` 标注（创作时标记，**仅标记位置，不进入最终台词**）。配音和字幕去除 `★` 符号但保留金句文本。
- 结尾段必须有互动钩子（直接向观众发问）
- **每句台词都要推动叙事**，禁止写纯信息陈述（如"他是一名电报员"→ 应写成"报务室里嘀嗒声不断，他双手在键钮上翻飞"）

---

**台词质量示例对比**：

| | 写法 | 问题 |
|:---:|------|:----:|
| ❌ 差 | 1912年4月14日，泰坦尼克号收到了冰山警告。电报员杰克很疲惫。 | 纯信息堆砌，无画面感，无张力 |
| ✅ 好 | 1912年4月14日深夜。泰坦尼克号最后的冰山警告，淹没在一堆付费电报里。首席电报员杰克·菲利普斯已连续工作了十几个小时——手指发麻，眼前发花，但他没停下，因为每一条发出去的私人电报，都是实打实的收入。 | 有场景、有感官细节、有矛盾（安全vs收入） |

| | 写法 | 问题 |
|:---:|------|:----:|
| ❌ 差 | 他按下按钮，发生了爆炸。 | 流水账，无情绪 |
| ✅ 好 | 他的手指悬在那个红色按钮上方，停顿了三秒。三秒里，他能听见自己的心跳盖过了所有的警报声。然后他按了下去。 | 有蓄势、有身体细节、有节奏感 |

---

**字数充实技巧**（用于每段台词自查）：

1. **加场景锚点**：不是"XX年发生了什么事"，而是"那个XX的夜晚/清晨/午后..."
2. **加身体细节**：不是"他感到很紧张"，而是"他指节发白"、"喉结上下滚动"、"手心全是汗"
3. **加因果链**：不止说"他做了X"，还要说"因为Y，所以他做了X"
4. **加对比反差**："所有人都以为X，但事实上Y"——这是故事性的核心

> ⚠️ 充实 ≠ 堆砌。每段台词控制在方案规定的字数以内。技巧是用**精准的细节替代空洞的陈述**，不是加更多废话。写完对照字数上限检查，超限优先精简，不超限无需刻意凑字。

### 3b. 图片生成指令（IMAGE PROMPT）

**格式规范**：
```
[主体：引用角色设定中的文化特征与视觉锚点] + [动作/状态] + [环境/背景] + [风格后缀] + [负向排除]
```

**要点**：
1. 每个出场核心角色的 prompt 必须包含文化/种族特征和视觉锚点英文描述
2. 多人场景差异化：每个角色含独特视觉锚点，标注"different person"
3. 具体胜过抽象，环境描写要丰富
4. 禁用动态镜头语言（不写"镜头推进""航拍"）
5. 负向排除默认加入 `no text, no letters, no watermark`；需要文字时移除
6. 英文输出

### 3c. 视频运动指令（VIDEO MOTION）

**格式规范**：
```
[camera movement] + [subject motion] + [environment motion] + [atmospheric elements]
```

**要点**：
1. 从分镜图出发，只描述运动和变化
2. 只描述物理运动，不描述叙事含义
3. 运动要克制，匹配历史纪录片节奏
4. 英文输出

**常用运动模式**：

| 情绪段 | 推荐运动 | 示例 |
|--------|---------|------|
| 冲击/钩子 | 极慢推进+主体微动 | subtle zoom in on clenched fist, dust floating |
| 压抑 | 水平缓移+环境静止 | slow horizontal pan, flags gently swaying |
| 荒诞 | 缓移+意外元素 | car slowly rolling to a stop, figure tensing |
| 暗涌 | 固定+光影变化 | flickering lamplight, shadows moving |
| 爆发 | 快速运动+混乱 | rapid motion, people rushing |
| 雪崩 | 宏大缓移+变化积累 | slow zoom out, shadows spreading across the map |
| 余韵 | 极慢+环境细节 | slow dim light shift, dust motes |

**输出**：追加到 `<projects_dir>/[项目名]/文案/创作底本.md`（接在视觉执行指令内容之后）

**⚠️ 本步产出格式（严格执行）**

```markdown
## 分镜脚本

**分镜规划说明**：
> 对节奏段的拆分规划进行说明。不要填入自估的「台词字数」表格。

**密度规划**：
> 分镜密度分配表（含配音模式、speed、emotion、intensity）

### 分镜 N / [节奏段名]-[镜头描述]

**配音模式**：有配音 / 无配音（二选一）
**配音指令**（仅配音模式=有配音时）：
> speed: 1.0 | emotion: calm | intensity: 0
> 气息标记：句中 (breath)，句尾 (exhale)
**预估配音时长**：（仅配音模式=有配音时）按公式 计算：`台词字数 / (3.5 × speed)`，向上取整为整数，范围3-15秒。若>15秒标注"超长，按15秒生成"
**目标时长**：（仅配音模式=无配音时）目标停留秒数，如 "5秒"
**台词**（配音模式=无配音时此处为空）：
> 具体台词内容（★标注金句位置，但★仅作创作标记）
> 使用 MiniMax 行内标签：句间换气用 (breath)，叹息用 (sighs)，吸气蓄势用 (inhale)，释重用 (exhale)

**角色参考图**：
> 本分镜出场核心角色的定妆参考图，格式：ref_角色ID1.jpg, ref_角色ID2.jpg

**IMAGE PROMPT**：
> 英文生图指令，一行写完

**VIDEO MOTION**：
> 视频运动描述，一行写完
```
