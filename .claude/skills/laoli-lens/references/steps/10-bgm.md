# 第10步：选取配乐

基于选题调性为视频选取贯穿全程的 BGM（纯音乐，一条到底）。提供三级策略，按优先级从高到低尝试。

**输入**：选题大纲（核心情绪/视觉风格/节奏段划分）

---

### 10a. 关键词搜索（推荐优先）

根据选题调性与视觉风格，生成中英文搜索关键词，供用户在免版权平台搜索 BGM。

- **BGM 方向**：基于选题大纲的核心情绪与视觉风格，推荐具体音乐风格和情绪关键词
- **示例**：`历史叙事 悬疑 管弦` / `historical narrative suspense orchestral`

### 10b. 人工生成

如果搜索未找到合适 BGM，生成 MiniMax 官网音乐生成页面可用的 prompt 词。

- **示例**：`管弦乐, 悬疑叙事节奏, 低音提琴与钢琴交替, 适合历史纪录片BGM`

### 10c. 自动生成（按需执行）

如果上述方式均不满足，经用户确认后，调用 `laoli-music` skill 基于 MiniMax API 生成 BGM。

- **调用**：`laoli-music` skill
- **prompt**：基于选题大纲生成音乐描述
- **输出目录**：`<projects_dir>/[项目名]/素材/BGM/`

---

**输出**：
- 10a/10b → 将关键词/prompt 保存到 `<projects_dir>/[项目名]/文案/配乐方案.md`
- 10c → 生成 `素材/BGM/bgm.mp3`，同时将描述存档到 `文案/配乐方案.md`

---
## ✅ 本步完成条件

配乐方案保存完成后，必须通过 Task tool 调用 `laoli-lens-reviewer` 进行评审（默认 L1：建议性文档格式校验）。仅当评审返回 **✅ 通过** 后，方可告知 Owner 本步完成、进入下一步。**未完成评审 = 本步未完成。**
